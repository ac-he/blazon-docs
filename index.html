Blazon is the language of heraldry.

Now it's a programming language too!

Drawing from the system heraldists have used for centuries to describe and define coats of arms, the Blazon Programming Language provides a unique opportunity to do two things at once: the blazon you write will be interpreted both as a program and as an armorial!

Every statement is structured as a blazon - a description of a particular coat of arms. The various divisions of the field represent different types of instruction and an assortment of supported charges represent different integers. These are used not only for their numerical values but also to label branches and name variables.
