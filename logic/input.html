Get User Input or Random Number

Per chevron [tincture 1] and [tincture 2]; in chief [charge 1], in base [charge 2].

or

Per chevron [tincture 1] and [tincture 2]; in chief [charge 1].

Example 1

Per chevron, purpure and or; in chief, keys saltirewise.

Read in a character and store its value to Variable107.

Example 2

Per chevron, sable and purpure; in chief a crown, in base a griffin.

Generate a random number 0-9 and store it to Variable33.

Tincture 1

Has no impact on execution. Consider taking this opportunity to make an aesthetic decision.

Charge 1

The integer-value of the VARIABLE being stored to.

Tincture 2

Determines which input action will be taken.
Get Input From Terminal
A Color
Get Random Number
A Metal
Input read in from the terminal must either be a whole number or a single character. Whole numbers will be interpreted as such, and characters will be converted into their integer values. Random numbers will be integers from 0 to 9.

Charge 2 [Optional]

Has no impact on execution. Consider taking this opportunity to make an aesthetic decision.
